# March Madness NCAA Basketball Website

This project is a React website built for IS 413. It displays NCAA basketball teams using data imported from a JSON file.

## Features

- Heading section introducing the page
- Team cards for each school
- Displays:
  - School name
  - Mascot name
  - Location (city, state)
- Team data loaded from a JSON file in the `src` folder
- Built with React, TypeScript, and Vite

## Components

- `Heading` – introduces the website
- `Team` – displays a single team card
- `TeamList` – renders all team cards from the JSON data
- `App` – combines all components

## How to Run

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open the local host link in your browser.

## Data Source

The team data comes from `CollegeBasketballTeams.json`, which is stored in the `src` folder.
